the .tre toolkit

A two-piece system for turning tree-shaped text into folders and back. One extension for the editor. One CLI for the terminal. Same format, both directions.

create.tre
vs code · ts

Right-click a .tre file or any selected tree text — folders and files exist on disk. Empty. Handles tree-drawing characters and Windows tree /F /A output. The editor-side half of the toolkit.

code > ext install polygonstew.create-tre
marketplace source
read.tre
cli · c#

The terminal-side half. Scaffold a tree from a .tre file, or capture an existing folder back into one. Same format as create.tre. Shipped via winget.

$ winget install polygonstew.read.tre
